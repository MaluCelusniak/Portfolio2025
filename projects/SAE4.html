SAE Semestre 4

Refonte d’une application Symfony / Java

Compétences : Symfony, Java Android, Docker, ergonomie, sécurité

Description : Refonte d’une application existante destinée à des professionnels. Site web Symfony + application mobile Java.

Travail Individuel :

  • - Mise en place d’un environnement Docker pour l’installation et le test
  • - Refonte de plusieurs interfaces web du site (back-office)

Le projet de la SAE du semestre 4 portait sur l’amélioration d’une application existante composée de deux éléments distincts mais interconnectés : une interface back-office web développée sous Symfony, destinée aux professionnels, et une application mobile Android en Java, destinée aux clients finaux. L’application souffrait à la fois de dysfonctionnements techniques et d’un manque d’ergonomie, rendant son utilisation peu intuitive.


Dans un premier temps, ma mission principale a été d’améliorer la sécurité et la portabilité du site web. Pour cela, j’ai conçu un environnement Docker capable de contenir l’application Symfony et ses dépendances, permettant un déploiement plus rapide, reproductible et sécurisé. Ensuite, je me suis concentré sur la refonte des pages du site web en travaillant sur leur ergonomie, en m’appuyant sur les critères de Bastien et Scapin, reconnus pour l’évaluation de l’utilisabilité d’une interface.


Ce projet m’a permis de consolider mes compétences en DevOps avec la création d’images Docker adaptées, mais aussi d’approfondir mes connaissances en Symfony et en conception d’interfaces web modernes. Le travail en équipe a également été important, car le projet nécessitait une coordination constante entre les développeurs du back-office et ceux de l’application mobile.
